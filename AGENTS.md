# LangChain Agent

A LangChain.js agent with tools (search engine and calculator) powered by LangGraph and the Vercel AI SDK.

## Tech Stack

- Next.js 15
- React 18
- LangChain.js
- LangGraph
- Vercel AI SDK
- OpenAI
- Tailwind CSS
